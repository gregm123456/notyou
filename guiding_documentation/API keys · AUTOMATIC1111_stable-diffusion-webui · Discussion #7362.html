      The alternative that I like to do is write a function something like

      def authenticate():
    s = requests.Session()
    s.auth = ('user', 'pass')
    return s

      Then any time you need to post or get, do s = authenticate_user() and all the requests afterward will be authenticated. Example:

      s = authenticate_user()

s.get(url + "/sdapi/v1/sd-models")
s.get(url + "/sdapi/v1/samplers")
s.get(url + "/sdapi/v1/prompt-styles")
s.get(url + "/sdapi/v1/face-restorers")
s.get(url + "/sdapi/v1/embeddings")
s.get(url + "/sdapi/v1/hypernetworks")
s.get(url + "/sdapi/v1/upscalers")
s.post(url=f'{url + "/sdapi/v1/options", json=model_payload)
s.post(url=f'{url + "/sdapi/v1/txt2img", json=payload)

      Either way works, but i find this to be a bit cleaner.
